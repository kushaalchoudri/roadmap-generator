Common Issues & Solutions

Issue 1: View Buttons Not Working

Symptoms: Clicking Month/Quarter/Year buttons does nothing
Check: Open console and look for "View buttons: null null null" or errors
Solution: Make sure you're on index.html?id=xxx (not home.html)

Issue 2: Today Marker Not Visible

Symptoms: No red line showing today's date
Check: Console should show "Today date:" and "Adding today marker at: Xpx"
Solution: Make sure your activities have dates close to today

Issue 3: Text Still Overlapping

Symptoms: Activity names and dates overlap
Check: Text padding increased to 180px in code
Solution: Try shorter activity names or switch to Quarter/Year view
